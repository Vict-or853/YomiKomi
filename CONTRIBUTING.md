# Colaborar con YomiKomi

Gracias por ayudar a mejorar YomiKomi. Este repositorio es informativo y no contiene el código fuente de la aplicación.

## Formas de colaborar

- Informar errores reproducibles.
- Proponer funciones o nuevos perfiles de dispositivos.
- Corregir errores de redacción en la documentación.
- Proponer traducciones para el contenido público.
- Mejorar la accesibilidad del sitio informativo.

## Antes de abrir una incidencia

1. Revisa las incidencias existentes para evitar duplicados.
2. Confirma que utilizas la versión más reciente disponible.
3. Reproduce el problema con una muestra que puedas compartir legalmente.
4. Elimina datos personales de capturas, rutas y registros.
5. Utiliza la plantilla adecuada y completa todos los campos requeridos.

## Cambios en la documentación

Las propuestas sobre archivos Markdown, plantillas de GitHub o el sitio ubicado en `docs/` son bienvenidas. Mantén cada cambio enfocado, explica el motivo e incluye una captura cuando afecte al diseño.

No envíes código de la aplicación, binarios modificados, claves, credenciales ni material protegido que no tengas permiso para distribuir.

## Conducta

Participa de forma respetuosa y constructiva. Los mensajes abusivos, el acoso, el spam y la publicación de datos personales pueden ser cerrados o eliminados.
