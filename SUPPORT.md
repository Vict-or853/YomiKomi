# Soporte

## Reportar un error

Utiliza la plantilla **Reporte de error** en [GitHub Issues](https://github.com/Vict-or853/yomikomi-tana/issues/new/choose). Incluye:

- Versión de YomiKomi.
- Edición de Windows y arquitectura.
- Formato de entrada y salida seleccionado.
- Perfil de dispositivo utilizado.
- Pasos exactos para reproducir el problema.
- Resultado esperado y resultado obtenido.
- Mensaje de error o captura, sin datos personales.

## Solicitar una función

Explica el problema que deseas resolver, cómo debería funcionar y qué dispositivos o formatos se beneficiarían. Para un nuevo perfil, incluye modelo exacto, resolución y fuentes públicas de sus especificaciones.

## Problemas con MOBI

Comprueba que Calibre y `ebook-convert`, o KindleGen, estén correctamente instalados y disponibles. Indica la versión de la herramienta utilizada al solicitar soporte.

## Contenido que no debes adjuntar

No compartas obras completas, archivos comerciales, credenciales, claves, datos personales ni información sensible. Utiliza contenido propio, libre o una muestra mínima autorizada.

## Vulnerabilidades

Los problemas de seguridad no deben publicarse como incidencias. Sigue las instrucciones de [SECURITY.md](SECURITY.md).
