# Privacidad

YomiKomi está diseñado para procesar localmente los archivos que seleccionas.

## Archivos procesados

- Las imágenes, cómics y libros se leen desde las ubicaciones que eliges en tu computadora.
- Los resultados se guardan en la ubicación seleccionada por el usuario.
- La conversión principal no requiere cargar el contenido a un servidor externo.

## Datos que debes proteger

Antes de enviar una incidencia o captura pública, elimina nombres de usuario, rutas personales, metadatos, títulos privados y cualquier contenido que no tengas permiso para compartir.

## Servicios externos

GitHub aplica sus propias políticas cuando visitas este repositorio, descargas una versión o creas una incidencia. Las herramientas opcionales de terceros, como Calibre o KindleGen, se rigen por sus propios términos y políticas.

## Cambios a este documento

Las modificaciones relevantes de esta política se publicarán en el historial del repositorio.
