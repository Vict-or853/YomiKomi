# Política de seguridad

## Versiones compatibles

La versión estable más reciente de YomiKomi recibe prioridad para correcciones de seguridad. Las versiones anteriores pueden dejar de recibir soporte cuando se publica una actualización.

## Informar una vulnerabilidad

No publiques vulnerabilidades, pruebas de concepto, datos personales ni rutas sensibles en una incidencia pública.

Utiliza la opción **Security → Advisories → Report a vulnerability** de este repositorio para enviar un reporte privado. Incluye:

- Versión afectada de YomiKomi.
- Edición y versión de Windows.
- Descripción del impacto.
- Pasos mínimos para reproducirlo.
- Archivos de prueba propios o libres, si fueran indispensables.
- Posibles mitigaciones conocidas.

Se acusará recibo cuando el reporte sea revisado. Los detalles deben permanecer privados hasta que exista una corrección o una mitigación pública.

## Alcance

Se consideran especialmente relevantes la ejecución inesperada de código, escritura fuera de la ubicación elegida, exposición de archivos locales, tratamiento inseguro de archivos comprimidos y manipulación de paquetes de actualización o descarga.
